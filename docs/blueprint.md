# **App Name**: ADL Analytics Hub

## Core Features:

- Entity Management: Create, read, update, and delete entities within the ADL system.
- Use Case Management: Create, read, update, and delete use cases associated with each entity.
- Metrics Dashboard: Display general, financial, business and technical metrics for each use case in separate tabs.
- Alerting System: Implement an alerting system, that alerts in function to an input

## Style Guidelines:

- Primary color: Main Action - #5303FF (Buttons, links, active states).
- Background color: White - #FFFFFF (Content area, cards, form fields).
- Neutral Gray 100: #E0E2E7 (Borders and Secondary Backgrounds).
- Neutral Gray 400: #858D9D (Secondary Text, labels).
- Neutral Black 800: #1A1C21 (Main Text, titles).
- Font: 'Poppins' (sans-serif) for a modern and readable interface. Note: currently only Google Fonts are supported.
- Display M (Page Title): 24px, Bold.
- Display S (Section Title): 20px, Bold.
- Text L (Main Body Text): 16px, Regular.
- Base Unit: 8px for all margins and padding.
- Main Page Padding: 32px on all sides.
- Card/Module Spacing: 24px.
